.Diseño del proyecto:

Para este proyecto se intenta emular un sitio web dedicado a la creacion y mantenimiento de paginas webs.
Se busco darle un estilo minimalista,para que sea agradable a la vista y mantener el foco de atencion en las busquedas centrales.

.Cambio Realizados:

1- Se agregan los enlaces "meta" con las "keywords" claves en cada pagina para optimizr la busqueda.
2- Se incorpora Boostrap en una de las paginas del proyecto (proyectos).
3- Se crea y utiliza los "mixins" y "extend",se logra una combinacion de ambos para la reutilizacion en los diseños
de las mismas.
4- Se elimina el "span" para darle mas uniformidad al contenido y que no quede tan chocante en el diseño.
5- Se incorporan "png" como logos de las tecnologias utilizadas y se realizan cambios sobre los mismos.
6- Se separa en la pagina principal las visualizaciones de la navegacion,haciendose 2(dos) estilos diferentes,
para cada una de las otras paginas del proyecto.
7- Se decide realizar el footer y la navegacion fusionados a la pagina,para que forme parte de la misma y genere un estilo visual
diferente.
8- Se utilizo una paleta de colores dentro de la gama de los grises, a exepcion de la pagina principal.
9- Se agrega el SiteMap al proyecto.
10- Se cambian titulos en pagina de NOSOTROS.
